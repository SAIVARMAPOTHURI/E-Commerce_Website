Step to Start Project
For Backend

git clone https://github.com/SAIVARMAPOTHURI/E-Commerce_Website
cd e-comerce-backend
npm i
npm run data:import
npm run dev
For Frontend

cd e-comerce-frontend
npm i
npm run start
